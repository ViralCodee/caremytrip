      SEO & AI / LLM

      Auto-generate llms.txt, sitemap.xml and robots.txt from your live packages & blog. Built for AI answer engines (ChatGPT, Claude, Perplexity, Gemini) and search crawlers.

      These files are generated automatically from your current data — including local admin edits. Files in a browser can't be written to the server directly, so after generating, download each file and commit it to your site root to publish.

      llms.txt AI

      An llms.txt manifest: company & Geo (Dehradun, Uttarakhand, India), SEO metadata, every tour package and every published blog post — so AI assistants can read and cite CareMyTrip accurately.
